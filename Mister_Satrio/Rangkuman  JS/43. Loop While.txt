

------------- JavaScript Sementara Loop -----------


The While Loop
---------------
Perulangan whilemelakukan perulangan melalui blok kode selama kondisi yang ditentukan benar.
Sintaksis
while (condition) {
  // code block to be executed
}
Contoh

Dalam contoh berikut, kode dalam perulangan akan dijalankan berulang kali selama variabel (i) kurang dari 10:
Contoh
while (i < 10) {
  text += "The number is " + i;
  i++;
} 

The Do While Loop
-----------------
Perulangan do whilemerupakan varian dari perulangan while. Perulangan ini akan mengeksekusi blok kode satu kali, sebelum memeriksa apakah kondisinya benar, kemudian akan mengulangi perulangan selama kondisinya benar.
Sintaksis
do {
  // code block to be executed
}
while (condition);
Contoh

Contoh di bawah ini menggunakan do whileloop. Perulangan akan selalu dieksekusi setidaknya satu kali, meskipun kondisinya salah, karena blok kode dieksekusi sebelum kondisi diuji:
Contoh
do {
  text += "The number is " + i;
  i++;
}
while (i < 10);
